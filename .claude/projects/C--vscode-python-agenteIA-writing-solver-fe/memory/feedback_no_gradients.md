---
name: No degradados
description: El usuario no quiere colores degradados (gradients) en la UI — usar colores solidos
type: feedback
---

No usar `linear-gradient`, `radial-gradient` ni ningun tipo de degradado en estilos CSS.

**Why:** Preferencia explicita del usuario por colores solidos y planos.

**How to apply:** En todo componente nuevo o existente, usar colores solidos (`background: #6200EA`) en vez de cualquier tipo de gradient.
